 # Playwright POM Login Demo 🚀

## 📌 Project Overview

This project is a simple example of a **login automation framework** built using **Playwright** and the **Page Object Model (POM)** design pattern.

The main goal of this project is to automate the login process of a sample web application while keeping the code clean, reusable, and easy to maintain. It follows good practices that are useful for real-world automation projects.

---

## 🛠️ Tech Stack

* **Language:** JavaScript
* **Framework:** Playwright
* **Design Pattern:** Page Object Model (POM)
* **Test Runner:** Playwright Test
